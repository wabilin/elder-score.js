# Not going to support

## ES6 (ES 2015) Supported Functions

We are not going to support functions already supported by ES6.

For example, `join`:

```js
_.join(['a', 'b', 'c'], '~'); // lodash
['a', 'b', 'c'].join('~'); // ES2105
```

- concat
- findIndex
- fill
- indexOf
- lastIndexOf
- join
- nth
- slice
